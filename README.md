# Deskripsi Singkat
QuizIClick adalah sebuah aplikasi kuis online berbasis web yang dibangun menggunakan PHP dan database MySQL/MariaDB. Tujuan utama aplikasi ini adalah untuk memungkinkan pengguna (siswa/peserta) untuk mengerjakan kuis/tes secara daring, serta memungkinkan pengguna (guru/admin) untuk membuat, mengelola soal dan kuis.

# Persyaratan Sistem
QuizIClick dibangun dengan PHP + css + javascript, persyaratan minimal:
1. Web server yang mendukung PHP yaitu XAMPP / WAMP / MAMP / LAMP (tergantung OS kamu)
2. PHP 
3. Database MySQL atau MariaDB
4. Browser web untuk mengakses aplikasi (Chrome, Firefox, dll)
# Langkah Instalasi
1. Download Project (ZIP)
- Buka repository QuizIClick di GitHub.
- Klik Code lalu Download ZIP.
- Ekstrak file ZIP.
2. Pindahkan ke folder XAMPP
- C:\xampp\htdocs\
3. Jalankan XAMPP
- Buka aplikasi XAMPP.
- Aktifkan Apache dan MySQL.
- Buka phpMyAdmin melalui browser.
- Buat database baru dengan nama projectexamrevisi.
- Pilih database, klik Import, lalu unggah file .sql.
4. Jalankan URL sesuai nama folder tempat kamu meletakkannya
- [http://localhost/QuizIClick](http://localhost/QuizIClick/)
